Ejercicio 3

Crear un modelo de objetos llamado piso que contenga las siguientes propiedades: metros cuadrados, altura, num habitaciones y precio, también se deben crear dos métodos: uno que muestre la información por pantalla y otro que calcule el IVA en función de un número que se introduzca por pantalla. Hacer un programa que permita crear un objeto de este tipo, solicitando todos los datos de las propiedades y el % de IVA que se aplica al precio. Finalmente se debe mostrar esa información por pantalla.
