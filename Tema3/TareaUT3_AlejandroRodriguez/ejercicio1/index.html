Ejercicio 1

Realiza un programa que cada 20 segundos (mediante setInterval) solicite un DNI hasta que alguien le escriba la cadena “-1”. En ese momento, el programa debe mostrar seguidas las letras de todos los DNIs introducidos. Aquí un enlace para saber como calcular la letra de DNI: https://es.wikipedia.org/wiki/N%C3%BAmero_de_identificaci%C3%B3n_fiscal
